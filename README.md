
Pocket Money Manager Pro is a modern full-stack web application designed to help parents and children manage pocket money efficiently. The platform enables parents to monitor allowance distribution, track spending behavior, and encourage responsible financial habits, while children can manage expenses, set savings goals, earn achievements, and improve money management skills.

Built with Python Flask, HTML, CSS, JavaScript, and SQLite, the application combines financial tracking, gamification, analytics, and AI-powered insights in an attractive Neon-themed user interface.

✨ Key Features
👨 Parent Dashboard
Monitor child spending
Manage allowance
View transaction history
Track goals and achievements
🧒 Child Dashboard
Record expenses
Track remaining balance
Create savings goals
View streaks and badges
💰 Allowance Management
Add and manage pocket money
Real-time balance updates
📊 Analytics & Reports
Spending insights
Expense visualization
Financial progress tracking
🏆 Gamification
Streak System
Achievement Badges
XP & Level Progression
Goal Completion Rewards
🤖 AI-Powered Financial Assistant (PocketAI)
Spending analysis
Savings recommendations
Budget suggestions
Goal forecasting
🔐 Security Features
User Authentication
Password Protection
Session Management
🛠️ Tech Stack
Backend: Python Flask
Frontend: HTML5, CSS3, JavaScript
Database: SQLite
Charts & Analytics: Chart.js
🎯 Project Objective

To promote financial literacy among children by providing an engaging platform that teaches budgeting, saving, and responsible spending while giving parents transparency and control over allowance management.
